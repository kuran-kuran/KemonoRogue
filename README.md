https://github.com/kuran-kuran/KemonoRogue/releases

# 遊び方
- タイトル画面でボタンを押したらゲームスタート。上下操作で音量を調節できます。  
- 方向キーでフレンズ操作して右ボタンで1ターン待機。セルリアンにぶつかると攻撃。
- 階段を降りると次のFloorへ。10階ごとにブラックセルリアン出現。全100階のランダムダンジョン
